# Support

## What to use for what

- Feature requests, non-sensitive bugs, and documentation improvements: GitHub issues
- Contribution proposals: GitHub issues or pull requests
- Security vulnerabilities: see [SECURITY.md](SECURITY.md)

## Before opening an issue

Please include:

- platform and version (`backend`, Android, or iPhone)
- deployment mode (`docker`, `docker compose`, reverse proxy, local backend, etc.)
- exact steps to reproduce
- expected behavior
- actual behavior
- relevant logs or screenshots

For self-hosting questions, include the relevant `.env` settings with secrets removed.

## Support boundaries

- This repository does not currently promise a response SLA.
- App Store / Play Store purchases do not create a separate commercial support contract.
- Official support guidance lives in the repository docs and issue tracker.
