# Trademarks and Branding

The source code in this repository is open-source under the repository license.

That does not grant rights to use the project's branding in a way that implies official affiliation.

## Reserved branding

The following are reserved for the official project and publisher distributions unless explicit permission is given:

- the `WakeFromFar` name when used as official product branding
- project logos and app icons
- official screenshots and store listing assets
- official release descriptions and marketing copy

## Forks and redistributions

If you fork or redistribute modified versions:

- do not imply endorsement or official status
- rename the app/project if needed to avoid confusion
- replace branding assets in redistributed binaries and store listings

This notice is intended to reduce user confusion around source forks and unofficial binaries. It does not require a registered trademark filing to communicate project branding expectations.
