Was ist WakeFromFar? What is WakeFromFar?

Ein privates Relay, das vollständig auf deiner eigenen Infrastruktur läuft – ohne fremde Server, ohne Datenübertragung in die Cloud. A private relay that runs entirely on your own infrastructure – no external servers, no data sent to the cloud.

So funktioniert's How it works

Drei Schritte zur vollständigen Kontrolle über dein Heimnetz. Three steps to full control over your home network.

VPN-Zugang erforderlich – nicht für offene Ports gedacht VPN access required – not designed for open internet ports

WakeFromFar bringt keine eigene VPN-Verbindung mit. Der Admin muss sicherstellen, dass das Backend ausschließlich über einen privaten Tunnel erreichbar ist – z.B. Tailscale, WireGuard oder einen Cloudflare Tunnel. Das Backend ist nicht dafür ausgelegt, mit offenen Ports im Internet betrieben zu werden. Nutzer brauchen Zugang zum selben Tunnel, um die App verwenden zu können. WakeFromFar does not include any VPN connectivity. The admin must ensure the backend is only reachable via a private tunnel – e.g. Tailscale, WireGuard, or a Cloudflare Tunnel. The backend is not designed to run with open ports exposed to the internet. Users need access to the same tunnel to use the app.

1

Tunnel einrichten & Backend deployen Set up tunnel & deploy backend

Richte einen privaten Tunnel ein (Tailscale, WireGuard oder Cloudflare Tunnel) und starte den Docker-Container auf deinem Heimserver. Das Backend darf nur über diesen Tunnel erreichbar sein – niemals mit offenen Ports ins Internet. Set up a private tunnel (Tailscale, WireGuard, or Cloudflare Tunnel) and start the Docker container on your home server. The backend must only be reachable through that tunnel – never with open ports to the internet.

2

Nutzer Tunnel-Zugang geben & App installieren Give users tunnel access & install app

Stelle sicher, dass deine Nutzer über denselben Tunnel erreichbar sind. Dann: WakeFromFar installieren, Backend-URL, Nutzername und Passwort eingeben – alles vom Admin bereitgestellt. Make sure your users are connected to the same tunnel. Then: install WakeFromFar, enter the backend URL, username, and password – all provided by the admin.

3

Maschinen aufwecken Wake your machines

Tippe auf ein Gerät in der App und wecke es mit einem Tap auf. Funktioniert von überall – solange der Nutzer mit dem Tunnel verbunden ist. Tap a device in the app and wake it with a single tap. Works from anywhere – as long as the user is connected to the tunnel.
