Was ist CarScreen? What is CarScreen?

Ein selbstgebautes Sekundärdisplay für Elektrofahrzeuge. Firmware für das Waveshare ESP32-P4 Round Display (800×800), das Fahrzeugdaten live über OVMS empfängt und mit LVGL visualisiert. A DIY secondary display for electric vehicles. Firmware for the Waveshare ESP32-P4 round display (800×800) that receives live vehicle data via OVMS and renders it with LVGL.

Funktionen Features

4 Themes4 Themes

NFSU2 (Rennsport-Neon), CYBER (blaues Glow), ANALOG (warmes Amber) und OEM. Per Swipe oder Menü wechselbar. NFSU2 (racing neon), CYBER (blue glow), ANALOG (warm amber), and OEM. Switchable via swipe or menu.

3 Dashboard-Modi3 Dashboard Modes

Standby (geparkt), Laden und Fahren – automatisch per State Machine umgeschaltet oder manuell überschrieben. Standby (parked), charging, and driving – automatically switched via state machine or manually overridden.

OVMS-IntegrationOVMS Integration

Live-Daten per WebSocket vom lokalen OVMS-Server: Geschwindigkeit, SOC, Temperaturen, Ladeleistung und mehr. Live data via WebSocket from the local OVMS server: speed, SOC, temperatures, charging power, and more.

Auto-HelligkeitAuto Brightness

VEML7700-Umgebungslichtsensor mit EMA-Glättung und logarithmischer Kurve. Manuelle Überschreibung im Menü. VEML7700 ambient light sensor with EMA smoothing and log curve. Manual override via menu.

Touch-GestenTouch Gestures

Swipe links/rechts zum Wechseln der Ansicht. Lang-Press (800 ms) öffnet das Quick-Menü. Fahrsperre bei Tempo > 5 km/h. Swipe left/right to cycle views. Long-press (800 ms) opens the quick menu. Menu locked at speed > 5 km/h.

Persistente EinstellungenPersistent Settings

Theme, Helligkeitsmodus und -bereich werden im NVS gespeichert und überleben Neustarts. Theme, brightness mode, and range are saved to NVS and survive reboots.

Hardware-Voraussetzungen Hardware Requirements

Das Projekt ist auf diese Hardware ausgelegt. Andere Boards erfordern Anpassungen. The project targets this specific hardware. Other boards require adaptation.

Waveshare ESP32-P4

ESP32-P4NRW32 mit rundem 800×800 IPS-Touchscreen (MIPI-DSI, JD9365). ESP32-C6 Koprozessor für Wi-Fi über SDIO. ESP32-P4NRW32 with round 800×800 IPS touchscreen (MIPI-DSI, JD9365). ESP32-C6 coprocessor for Wi-Fi via SDIO.

OVMS

Open Vehicles Management System – lokal im Fahrzeugnetzwerk erreichbar. CarScreen verbindet sich per WebSocket mit dem OVMS Web-UI. Open Vehicles Management System – reachable locally in the vehicle network. CarScreen connects to the OVMS Web UI via WebSocket.

VEML7700 (optional)

I²C-Umgebungslichtsensor für automatische Helligkeitsregelung. Ohne den Sensor läuft das System im manuellen Modus. I²C ambient light sensor for automatic brightness control. Without it, the system runs in manual brightness mode.
