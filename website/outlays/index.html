Was Outlays kann What Outlays does

Automatische SynchronisationAutomatic Sync

Einmal anmelden, dann läuft der Sync im Hintergrund. OAuth PKCE für Lidl Plus, Playwright für session-basierte Händler. Keine manuelle Token-Kopiererei. Authenticate once, sync runs in the background. OAuth PKCE for Lidl Plus, Playwright for session-based retailers. No manual token copying.

Rabatt-IntelligenzDiscount Intelligence

Drei Rabatttypen auseinandergehalten: Lidl Plus Mitgliedsrabatte, reguläre Aktionen und MHD-Rabatte. Keine zusammengefassten Totals – echte Transparenz. Three discount types distinguished: Lidl Plus member discounts, regular promotions, and best-before discounts. No collapsed totals – real transparency.

Analytics DashboardAnalytics Dashboard

Monatliche Ausgaben, Top-Händler, Kategorien, Sparquote und Trends. Wiederkehrende Ausgaben werden automatisch erkannt. Monthly spend, top stores, categories, savings rate, and trends. Recurring purchases are detected automatically.

Agent-ready APIAgent-ready API

Vollständige OpenClaw-kompatible API – dein KI-Agent kann Ausgaben abfragen, Muster erkennen und Anomalien melden. Nicht nur Diagramme ansehen, sondern aktiv analysieren lassen. Full OpenClaw-compatible API – your AI agent can query spending, detect patterns, and flag anomalies. Not just looking at charts, but actively having them analysed.

Desktop AppDesktop App

Electron-App für macOS und Windows mit gebündeltem Backend. Kein Docker nötig für Nicht-Techniker. Backup und Restore per Klick. Electron app for macOS and Windows with bundled backend. No Docker required for non-technical users. One-click backup and restore.

Komplett privatCompletely private

Alle Daten bleiben bei dir. SQLite lokal, kein Cloud-Zwang, keine Telemetrie. Selbst gehostet via Docker oder direkt als Desktop-App. All data stays with you. SQLite locally, no cloud dependency, no telemetry. Self-host via Docker or run directly as a desktop app.

Gebaut für KI-Agenten. Built for AI agents.

Outlays implementiert das OpenClaw-Protokoll – eine strukturierte API, über die KI-Agenten direkt auf deine Ausgaben zugreifen, Muster analysieren und Fragen beantworten können. Frag deinen Agenten: „Was habe ich letzten Monat bei REWE ausgegeben?" – und bekomm eine echte Antwort aus deinen eigenen Daten. Outlays implements the OpenClaw protocol – a structured API that lets AI agents directly access your spending, analyse patterns, and answer questions. Ask your agent: "What did I spend at REWE last month?" – and get a real answer from your own data.
